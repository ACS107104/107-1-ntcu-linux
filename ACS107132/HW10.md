###1.請仿照課堂上練習，透過systemd管理二個sshd服務，並讓第二個sshd服務的 port 放行於 2222。

1. **systemctl status sshd**查看sshd的狀態 [1.png]
2. **yum provides semanage**安裝
3. **yum install policycoreutils-python**安裝
4. **cd /etc/ssh**-->**cp sshd_config sshd2_config**
5. **vi sshd2_config**更改內容 [2.png]
6. **cd /etc/systemd/system**-->**cp /usr/lib/systemd/system/sshd.service sshd2.service**
7. **vi sshd2.service**更改內容 [3.png]
8. **semanage port -a -t ssh_port_t -p tcp 2222**修改port-->**semanage port -l | grep ssh**查看 [4.png]
9. **systemctl daemon-reload**讀取-->**systemctl start sshd2.service**-->**systemctl status sshd2**
