1.開啟虛擬器，登入root，然後輸入systemctl status sshd就能檢查目前sshd系統服務狀態(圖1)，
再輸入cd /etc/ssh，進入到ssh的目錄，輸入cp sshd_config sshd2_config將sshd的內容複製到sshd2，
然後輸入vi sshd2_config，在按i進入編輯，在port的位置改成2222(圖2)，輸入wq跳出編輯。
再輸入cd /etc/systemd/system切回system的目錄，輸入cp /usr/lib/systemd/system/sshd.service sshd2.service
將/usr/lib/systemd/system/sshd.service複製到sshd2.service。輸入vi sshd2.service編輯，
在description那行的最後加上2，
還要在ExeStart輸入-f /etc/ssh/sshd2_config改掉之前的/usr/sbin/sshd(圖3)。
2.輸入systemctl daemon-reload來重讀設定檔，再輸入systemctl enable sshd2讓下次開機時，
sshd2可以被啟動，輸入systemctl start sshd2，來啟動sshd2，但出現fail(圖4)，
所以先安裝semanage，輸入yum install policycoreutils-python，安裝完成後(圖5)，
接著輸入semanage port -a -t ssh_port_t -p tcp 2222，讓ssh可以使用2222連接阜號(圖6)，
再輸入一次systemctl start sshd2就能成功啟動sshd2了(圖7)，
輸入netstat -tlnp |grep ssh來確認是否有啟動兩個sshd服務。
