#CentOS Download and Install
<ol>
<li>下載CentOS iso檔</li>
<li>開啟VirtualBox</li>
<li>按下新增</li>
<li>輸入自訂名稱</li>
<li>輸入或調整RAM大小</li>
<li>選擇"立即建立虛擬硬碟"</li>
<li>選擇VDI</li>
<li>選擇"固定大小"</li>
<li>輸入或調整虛擬硬碟大小</li>
<li>右鍵點擊選擇設定值</li>
<li>點擊左方"存放位置"</li>
<li>點擊光碟圖示，選擇下載下來的CentOS iso檔</li>
<li>點擊左方"網路"</li>
<li>選擇"介面卡2"並啟用</li>
<li>附加到"「僅限主機」介面卡"</li>
<li>按下進階並將混和模式調成"允許全部"</li>
<li>按下確定</li>
<li>開啟虛擬機器</li>
<li>選擇Install CentOS</li>
<li>選擇語言</li>
<li>選擇"安裝目的地"</li>
<li>選擇"讓我自行配置磁碟分割"</li>
<li>選擇"標準分割區"，按下+分配磁碟區，按下確定</li>
<li>設定ROOT密碼及建立用戶，等待設定完成</li>
<li>結束設定</li>
