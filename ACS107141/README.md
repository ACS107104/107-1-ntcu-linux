##2

*root和同群組的帳號具有rwx權限，其他帳號則只有rx權限

1.
```
#chmod 777 ~/script.sh
```
2.
```
#chmod u=rwx,g=rwx,o=rwx ~/script.sh
```

##3

*實體連結是在目錄裡加一筆inode與檔名對應
*符號連結是建立獨立檔案指向目的檔案

*實體連結
1.
```
#ll -i /etc/hosts
``` 
2.
```
#ln /etc/hosts .
```
3.(檢查) 
```
ll -i /etc/hosts hosts.real
```

*符號連結
1.
```
#ln -s /etc/hosts hosts.symbo
```
2.(檢查) 
```
#ll -i /etc/hosts /root/hosts.symbo
```
