##HW-3

<ol>
<ls>輸入"ls -ali /etc/hosts"</ls>
![](https://i.imgur.com/u03xGHq.png)
>最前面的數字為inode
>1個檔名在使用</ls> 
</ol>

<ol>
<ls>輸入"ln /etc/hosts /srv/hosts.hard"建立實體連結</ls>
<ls>輸入"ls -ali /srv/hosts.hard"</ls>
![](https://i.imgur.com/AAzOcjO.png)
>最前面的數字為inode
>2個檔名在使用</ls>
*/etc/hosts和建立的實體連結/srv/hosts.hard都在使用，所以有兩個檔名
</ol>

<ol>
<ls>輸入"ln -s /etc/hosts /srv/hosts.soft"建立符號連結</ls>
<ls>輸入"ls -ali /srv/hosts.soft"</ls>
![](https://i.imgur.com/n2NKr6b.png)
>最前面的數字為inode
*符號連結為獨立的檔案，所以不會像實體連結一樣共用同一個inode
</ol>
