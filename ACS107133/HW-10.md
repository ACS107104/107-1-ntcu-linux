1.輸入systemctl status sshd 查看sshd的狀態(圖1)

2.yum install vsftpd 安裝

3.yum provides semanage安裝，輸入yum install policycoreutils-python安裝

4.輸入cd /etc/ssh，輸入cp sshd_config sshd2_config

5.輸入vi sshd2_config 更改內容（圖2)

6.輸入cd /etc/systemd/system ，輸入cp /usr/lib/systemd/system/sshd.service sshd2.service複製

7.輸入vi sshd2.service 更改內容(圖3)

8.輸入semanage port -a -t ssh_port_t -p tcp 2222更改port，輸入semanage port -l | grep ssh查看(圖4)

9.輸入systemctl daemon-reload讀取，輸入systemctl start sshd2.service開啟sshd2_service

10.輸入netstat -alntp | grep ssh看結果(圖5)
